# Fiverr Clone (Frontend)

A basic **Fiverr-style UI clone** built with **React + Vite**.
This is a frontend project focused on layout, routing, UI states, and reusable components (not a real Fiverr backend).

## Tech

- React + Vite
- React Router
- Redux Toolkit
- Axios
- Formik
- i18next
- Radix Dialog
- SweetAlert2
- lucide-react

## Commands

Install:

npm install

Run dev:

npm run dev

Lint:

npm run lint
